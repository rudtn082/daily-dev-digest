## 这是什么

## 今天 — 2026-07-21

智能体栈走向成熟之日 —— 与其只是生成更多代码和 UI，今天的工具在模型产物周围加上护栏与协同；同时一款不依赖 NVIDIA 芯片训练的万亿参数开源模型登场。

| 精选 | 是什么 | 为何引起注意 |
|------|-----------|--------------|
| **Hallmark** | 面向 Claude Code、Cursor、Codex 的反 AI-slop 设计技能 | 植入拒绝“每个模型都学成的紫色渐变默认值”的规则，在交回 UI 前先跑一批 slop-test 关卡和自我批评 —— 对“AI 设计千篇一律”的具体回应 |
| **Aura** | AI 编码智能体的 git 原生溯源层 | 在本地把仓库解析成 AST，按函数级追踪智能体的改动，与你声明的意图对比，不一致就拦下提交 —— 还能只 `rewind` 一个坏掉的函数，并清除 AI 的幻觉记忆 |
| **Mozaik** | 让智能体自组织的 TypeScript 运行时 | 抛弃预先画好的 DAG：每条消息、工具调用、错误都是共享总线上的事件，智能体只订阅并响应自己关心的 —— 协同在运行时涌现，多个智能体同时朝同一目标推进 |
| **LongCat-2.0** | 全程用中国芯片训练的 1.6T 开源 MoE | 万亿规模却是 MIT 许可，原生 1M token 上下文 —— 美团称这是同规模中首个预训练与推理全部在国产 AI ASIC（而非 NVIDIA GPU）上完成的模型 |

## 今天 — 2026-07-20

反黑箱之日 —— 与其信任厂商的运行时，或在看不见的地方触发副作用的逻辑，这批工具把智能体的源码直接交给你、让持久工作流跑在你早已运行的 Postgres 上、把 I/O 变成可读的普通数据，还让你自己托管边缘平台。

| 精选 | 是什么 | 为何引起注意 |
|------|-----------|--------------|
| **Grok Build** | 已开源的 xAI 编码智能体 CLI | TUI、智能体运行时与 ACP 接线的真实 Rust 源码以 Apache-2.0 公开 —— 直接读懂前沿实验室如何组装上下文、如何调度工具（Issue 关闭、拒绝外部 PR，是源码释出而非社区项目） |
| **DBOSify** | 建在 Postgres 上的 Temporal 直替方案 | 把 `temporalio` 换成 `dbosify` 即可获得持久工作流、重试与恢复，无需另跑服务器 —— 移植了 Temporal 自己的测试来验证兼容性，可自行核查 |
| **Pure Effect** | 变成可回放普通数据的业务逻辑 | 函数不执行 I/O，而是返回描述“打算做的 I/O”的对象 —— 记录生产中所见，在你的笔记本上回放同一条路径，完全不碰数据库 |
| **OpenWorkers** | 自托管的 Cloudflare Workers，用 Rust 写成 | 在你自己机器上运行的 Workers 编程模型 —— `fetch`、KV、Postgres 绑定、S3/R2、cron、`crypto.subtle` —— 当边缘的便利撞上锁定与出网流量账单时的退路 |

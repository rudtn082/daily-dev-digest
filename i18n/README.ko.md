## 오늘 — 2026-07-22

에이전트의 배관이 진지해지는 날. 더 많이 뽑아내는 대신, 오늘 픽들은 안쪽을 향한다 — 에이전트를 관측 가능하게 만들고, 기억을 주고, 내 기계에서 돌리고, 보안에 겨눈다.

| 픽 | 뭔데 | 왜 눈에 띔 |
|------|-----------|------------|
| **mcpsnoop** | "MCP용 Wireshark" — 툴 호출을 들여다보는 투명 프록시 | MCP Inspector는 자기 클라이언트로 붙어 채널 *옆*에 서지만, mcpsnoop은 실제 파이프 안에 끼어듦 — 프로덕션 클라이언트와 서버가 실제 주고받는 JSON-RPC, 서버 stderr, 느리거나 멈춘 호출을 그제야 본다 |
| **Strix** | 침투 테스트용 오픈소스 AI 에이전트 | 정적 분석의 오탐 대신 앱을 동적으로 돌려 각 발견을 실제 PoC로 검증, 전문화된 에이전트를 타깃마다 병렬로 뿌림 — 공격을 일급 자율 워크플로로 |
| **Juggler** | JUCE 제작자가 만든 오픈소스 GUI 코딩 에이전트 | CLI 에이전트 물결을 거슬러 그래픽 UX로 감 — UX에 진짜 내공 있는 베테랑 프레임워크 저자가 "터미널은 코딩 에이전트의 집이 아니다"라고 주장하는 셈이라 지켜볼 만 |
| **LM Studio Bionic** | 채팅창이 아닌, 오픈 모델을 위한 완전한 로컬 에이전트 | 채팅을 넘어 문서를 편집하고, 인라인 diff로 에이전틱 코드 검색을 돌리고, 온디바이스 음성 전사까지 — 무거운 작업만 클라우드로 보냄, 데이터 미보존 로컬 퍼스트 |
| **TencentDB-Agent-Memory** | 외부 API 없는, 에이전트용 완전 로컬 장기 기억 | 4단계 로컬 파이프라인(로컬 LLM + SQLite 벡터)이 툴 로그를 압축 심볼로 응축하고 대화를 구조화된 페르소나·장면으로 증류 — 평평한 벡터 더미 대신 |

## 오늘 — 2026-07-23

물결은 계속 *새 에이전트를 만들어라*라고 말한다. 오늘 픽들은 반대다 — 이미 돌리는 에이전트에 하네스를 씌워라. 조율하고, 압박 테스트하고, 캔버스에 배선하고, 도메인에 겨눠라.

| 픽 | 뭔데 | 왜 눈에 띔 |
|------|-----------|------------|
| **T3MP3ST** | 이미 쓰는 코딩 에이전트를 몰고 다니는 자율 레드팀 메타 하네스 | 자체 모델을 안 실음 — 로그인해 둔 에이전트(Claude Code, Codex, 완전 오프라인 모델)를 감싸 정찰 → 익스플로잇 → 리포트 킬체인을 돌리고, egress-scope 봉쇄로 범위 밖 호스트를 자동 거부; 레드팀을 하네스 문제로 재정의했고 이번 달 빠르게 치고 올라옴 |
| **Sim** | 에이전트 워크플로를 짜고 조율하는 오픈소스 시각 캔버스 | 붐비는 판이지만 `npx simstudio` 한 줄로 셀프호스팅되고 전부 Ollama에 로컬로 붙어 외부 API가 없다 — 원커맨드·로컬 퍼스트라는 점이 Product Hunt 첫 화면까지 밀어올림 |
| **OpenScience** | 연구 루프 전체를 도는 모델 무관 워크벤치 | 목표를 주면 논문을 읽고, 가설을 세우고, 실제 컴퓨트에서 코드를 짜 돌리고, 과학 DB를 질의하고, 결과를 써낸다 — 브라우저 한 세션에서 연속으로, Apache-2.0, BYOK, Claude Science의 독립 대안임을 공개적으로 표방 |
| **agent-drift** | 코딩 에이전트가 압박에 무너지는지 압박 테스트 | ICLR 2026 "Asymmetric Goal Drift" 결과를 실행 가능하게 만듦: 제약이 보안 같은 학습된 가치와 충돌할수록 에이전트가 시스템 프롬프트를 *더* 어기고, 코드 속 평범한 주석만으로도 시간이 지나면 지시를 덮어씀 — 이제 내 에이전트에서 재현 가능 |

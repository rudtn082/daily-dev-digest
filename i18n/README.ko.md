## 오늘 — 2026-07-21

에이전트 스택이 성숙하는 날 — 코드와 UI를 더 많이 뽑아내기만 하는 대신, 오늘 도구들은 모델이 만든 결과물 주위에 가드레일과 조율을 두른다. 그리고 NVIDIA 실리콘 없이 학습된 조 단위 파라미터 오픈 모델이 등장한다.

| 픽 | 뭔데 | 왜 눈에 띔 |
|------|-----------|------------|
| **Hallmark** | Claude Code·Cursor·Codex용 안티-AI-슬롭 디자인 스킬 | 모든 모델이 학습해버린 보라색 그라디언트 기본값을 거부하는 규칙을 심고, UI를 넘기기 전에 슬롭 테스트 게이트와 자기비평 패스를 돌림 — "AI 디자인은 다 똑같다"에 대한 구체적 답 |
| **Aura** | AI 코딩 에이전트용 git 네이티브 프로버넌스 레이어 | 리포를 로컬에서 AST로 파싱해 에이전트의 수정을 함수 단위로 추적, 명시한 의도와 대조해 어긋나면 커밋을 막음 — 망가진 함수 하나만 `rewind`하고 AI의 환각 기억을 지우는 기능까지 |
| **Mozaik** | 에이전트가 스스로 조직화되는 TypeScript 런타임 | 미리 그리는 DAG를 버림: 모든 메시지·툴 호출·에러가 공유 버스 위 이벤트가 되고 에이전트는 관심 있는 것만 구독·반응 — 조율이 런타임에 창발하며 여러 에이전트가 같은 목표에 동시 작업 |
| **LongCat-2.0** | 중국산 칩만으로 학습한 1.6T 오픈 MoE | 조 단위 규모에 MIT 라이선스, 네이티브 1M 토큰 컨텍스트 — Meituan은 이 규모에서 사전학습과 추론을 전부 국산 AI ASIC(NVIDIA GPU 아님)으로 끝낸 첫 모델이라 밝힘 |

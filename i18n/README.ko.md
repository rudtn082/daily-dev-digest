## 오늘 — 2026-07-20

블랙박스 거부의 날 — 벤더 런타임이나 보이지 않게 부작용을 터뜨리는 로직을 믿는 대신, 에이전트의 소스를 그대로 넘겨주고, 오래가는 워크플로를 이미 굴리는 Postgres 위에 얹고, I/O를 읽을 수 있는 평범한 데이터로 만들고, 엣지 플랫폼까지 직접 호스팅하게 해준다.

| 픽 | 뭔데 | 왜 눈에 띔 |
|------|-----------|------------|
| **Grok Build** | 오픈소스로 풀린 xAI 코딩 에이전트 CLI | TUI·에이전트 런타임·ACP 배선의 실제 Rust 소스가 Apache-2.0으로 공개 — 프런티어 랩이 컨텍스트를 어떻게 조립하고 툴을 어떻게 호출하는지 그대로 읽을 수 있음 (이슈 비활성·외부 PR 거부, 참여용이 아니라 소스 공개) |
| **DBOSify** | Postgres 위에 얹는 Temporal 드롭인 대체 | `temporalio` 대신 `dbosify` 임포트만으로 내구성 워크플로·재시도·복구 확보, 별도 서버 불필요 — Temporal 자체 테스트를 포팅해 호환성을 직접 확인 가능 |
| **Pure Effect** | 재생 가능한 평범한 데이터가 된 비즈니스 로직 | 함수가 I/O를 실행하는 대신 "하려던 I/O"를 기술하는 객체를 반환 — 운영에서 본 걸 기록해 노트북에서 동일 경로 재생, DB는 건드리지 않음 |
| **OpenWorkers** | 셀프호스팅 Cloudflare Workers, Rust로 | 내 서버에서 돌리는 Workers 프로그래밍 모델 — `fetch`·KV·Postgres 바인딩·S3/R2·cron·`crypto.subtle` — 엣지 편의가 락인·이그레스 비용과 부딪힐 때의 탈출구 |

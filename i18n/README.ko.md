## 오늘 — 2026-07-18

무거운 작업이 다시 내 하드웨어로 — 진짜 Office 파일을 편집하는 에이전트, 수백만 점을 그리는 GPU, 기기 안에서 받아쓰는 회의, 그리고 내 것을 감사·보관하는 도구들.

| 픽 | 뭔데 | 왜 눈에 띔 |
|------|-----------|------------|
| **OfficeCLI** | AI 에이전트가 굴리는 Office 스위트 | 단일 바이너리, MS Office 불필요 — 에이전트가 Word·Excel·PowerPoint를 실제 수식 엔진과 렌더-수정 루프로 읽고 편집 |
| **Kage** | 웹사이트를 오프라인 단일 바이너리로 그림자 복제 | DOM 스냅샷 후 JS 전부 제거, CSS·이미지를 하나의 실행파일로 묶어 오프라인 서빙 — 받는 쪽엔 아무것도 설치 안 함 |
| **Meetily** | 셀프호스트·100% 로컬 AI 회의 노트 | Rust + Tauri, Parakeet/Whisper 실시간 전사·화자 분리·Ollama 요약 — 기기 밖으로 안 나가는 오픈 Otter/Granola |
| **Tailsnitch** | Tailscale 테일넷 보안 감사기 | 접근·인증·기기·네트워크·SSH·DNS 걸친 50여 개 점검, 수정 모드와 SOC 2 매핑 — "여기 너무 열어둔 거 아냐?"용 린터 |
| **ChartGPU** | 백만 점에도 매끄러운 WebGPU 차트 | 파이프라인 전체를 GPU로 — 다운샘플링 컴퓨트 셰이더·인스턴스 드로우 — 100만 점 60fps 팬/줌, 벤치는 수천만 점 |

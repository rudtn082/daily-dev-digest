## 오늘 — 2026-07-19

에이전트를 내 하드웨어에서 돌리기 위한 뼈대가 채워지는 중 — 샌드박스로 가두고, 터미널 하나로 몰아 부리고, 오래가는 기억을 붙이고, 그걸 돌리는 서버까지 관리한다.

| 픽 | 뭔데 | 왜 눈에 띔 |
|------|-----------|------------|
| **CubeSandbox** | AI 에이전트용 즉석 보안 샌드박스 | 60ms 미만 부팅, 인스턴스당 5MB 미만 — 한 노드에 수천 에이전트, 값싼 격리가 플릿의 채산성을 가르는 지점 |
| **herdr** | 터미널에 사는 에이전트 멀티플렉서 | 에이전트판 tmux — 여러 코딩 에이전트를 동시에 띄우고 키보드로 전환, 무거운 데스크톱 앱도 중간 과금도 없음 |
| **Statewave** | 재현성 위에 세운 에이전트 메모리 런타임 | 쿼리 시점 RAG 대신 출처 태그가 붙은 컨텍스트 번들 — 모든 기억이 근거로 추적되고 감사 로그가 남아, 에이전트가 뭘 알았는지 디버깅 가능 |
| **Alien** | 원격 관리 내장 셀프호스팅, Rust로 | 소프트웨어는 내 환경 안에서 데이터를 밖에 안 내보내며 돌고, 그 서버들은 원격으로 깔끔히 관리 — 온프렘 마찰이 실제로 터지는 지점 |

# Daily Dev Digest

開発とAI全般で、一度見ておく価値のある新しいツール・リポジトリ・アイデアを毎日短くまとめた一覧。
1日に数件、それぞれ「なぜ目に留まったか」を1行で。4言語で。

## これは何

毎日流れてくる量は追いきれない。これはそのフィルター — 1日3〜5件、それぞれなぜクリックする価値があるかを1行で。
テーマは固定しない。エージェント、インフラ、小さいが賢いツール、モデル、たまに風変わりな実験も。
その日の一覧はアーカイブにそのまま凍結 — 後からこっそり書き換えたりしない。

掲載したものとは無関係。ここにあるのは推薦ではなくポインタ — 自分で見て判断を。

## 選び方

- 1日3〜5件、多くて5件。目立つものがない日は少なめに。
- 重複なし — アーカイブにあるものと照合。
- 皆が知る巨大プロジェクトより、伸びているものを優先。
- 有料枠・アフィリエイトリンクなし、例外なし。

提案したい、または自分で複製して回したいなら [`CONTRIBUTING.md`](../CONTRIBUTING.md)。

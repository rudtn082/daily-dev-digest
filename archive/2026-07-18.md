# Daily Dev Digest — 2026-07-18

> Theme: heavier work moves back onto hardware you own — agents that edit real Office files, a GPU rendering millions of points, meetings transcribed on-device — plus tools to audit and archive what's yours.

## Picks

### OfficeCLI — an Office suite built for AI agents to drive
A single-binary, open-source CLI (no Microsoft Office install needed) that lets agents read, edit,
and automate Word, Excel and PowerPoint through a DOM-like path system — with a native formula
engine (350+ Excel functions) and an HTML/PNG render loop so the agent can look at what it made and
fix it. It installs into Claude Code, Cursor, Windsurf and Copilot. Most "agents + documents" demos
stop at plain text; this is a serious attempt to give them real office file formats.

### Kage — shadow any website into a single offline binary
Kage (影, "shadow") drives real headless Chrome, snapshots the DOM a human would have seen, then
strips every script, event handler and `javascript:` URL and pulls CSS, images and fonts down to
local paths. The `--format binary` option glues the archive onto a copy of itself, handing you one
executable that serves the site offline with nothing installed on the other end. A clean, inert take
on personal archiving in a web that assumes you're always online.

### Meetily — self-hosted, 100% local AI meeting note-taker
A Rust + Tauri app (macOS/Windows) doing live transcription with Parakeet/Whisper, speaker
diarization and Ollama summarization — all on-device, no cloud. Notable as an open, self-hostable
answer to Otter/Granola/Fireflies for anyone who can't send meeting audio off their machine, and a
good showcase of how far local speech + LLM stacks have come.

### Tailsnitch — a security auditor for your Tailscale tailnet
Scans a tailnet against 50+ checks across access control, auth, device hygiene, network exposure,
SSH, logging and DNS, with an interactive fix mode, SOC 2 / Common Criteria mappings, and JSON
output for CI. As mesh VPNs quietly become the backbone of small-team infra, a dedicated linter for
"did we leave this too open?" is exactly the missing guardrail.

### ChartGPU — WebGPU charting that stays smooth at a million points
An MIT-licensed TypeScript library that moves the whole pipeline onto the GPU — LTTB downsampling as
a compute shader, GPU-accelerated hit-testing, one instanced draw call per series — so 1M points pan
and zoom at 60fps and benchmarks push tens of millions. For anyone who's watched a dashboard choke on
a big time series, it's a concrete look at what WebGPU unlocks for everyday dataviz.

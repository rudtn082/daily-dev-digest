# Daily Dev Digest — 2026-07-16

> Theme: the agentic coding stack keeps consolidating around MCP, local-first memory & sandboxing.

## Picks

### orca — parallel coding-agent dev environment
An environment for running multiple coding agents in parallel with proper observability. It packages the
orchestration + memory + execution layer that people kept hand-wiring around single agents.

### graphify — codebase → queryable knowledge graph
Turns a repository into a knowledge graph you (or an agent) can query, giving structured context instead of
raw file dumps. Useful for large-repo reasoning and retrieval.

### Browser Use — LLM agents that drive a real browser
A clean, fast-growing take on giving agents real browser control for web automation.

### Unsloth — faster, cheaper LLM fine-tuning
~2× faster fine-tuning with lower VRAM, making local / consumer-GPU fine-tuning practical.

### pocket-tts — CPU text-to-speech
A text-to-speech model that runs on CPU — local-first, no GPU tax, good for offline and edge use.

# Daily Dev Digest — 2026-07-17

> Theme: the layers around the AI coding agent are filling in — orchestration, grounding, a tiny local brain, a voice, and supply-chain safety.

## Picks

### Orca — agent development environment for a fleet of parallel coding agents
An open-source (MIT), YC-backed desktop+mobile app that runs 30+ CLI agents — Claude Code, Codex,
Cursor CLI, Copilot CLI and more — side by side in isolated git worktrees, on your own subscriptions
with no middleman billing. As people juggle several agents at once, the missing piece has been a
place to launch, review, and steer them; Orca is a credible take on that control room.

### Graphify — turn a whole codebase into a queryable knowledge graph
A skill for Claude Code, Cursor, Codex, Gemini CLI and others (YC S26) that parses code, SQL schemas,
docs and more into one graph using tree-sitter AST — deterministic, on-device, no embeddings and no
vector store. It targets the real weakness of coding assistants: reasoning across a large project
instead of one file at a time, and it does it without shipping your code anywhere.

### Needle — Gemini tool-calling distilled into a 26M-parameter model
A tiny function-calling model from Cactus Compute built on the argument that tool calling is
retrieval-and-assembly, not reasoning — so the architecture is pure attention with no MLPs, and it
runs fast enough for phones and watches. It hit #1 on Show HN and is a sharp counter to the
"bigger model for everything" default: for structured tool use, small and specialized may be plenty.

### Voicebox — open-source, local-first AI voice studio
From the creator of Spacedrive: hold a key, speak, and the transcript drops into whatever field has
focus — with voice cloning, multi-engine TTS across 23 languages, and a timeline editor, all running
locally via Whisper on Apple Silicon or GPU. Notable as a voice-in/voice-out loop for agent work that
stays entirely on your machine, no subscription and no cloud round-trip.

### Bumblebee — read-only supply-chain scanner for developer machines
Perplexity's Go tool (Apache-2.0, zero non-stdlib deps) inventories what's actually installed —
npm/PyPI/Go/RubyGems/Composer packages plus MCP configs and editor/browser extensions — and flags
matches against a known-compromise catalog, without ever executing a package or reading source. After
a year of noisy package-registry compromises, a check that refuses to run your code is the right shape.
